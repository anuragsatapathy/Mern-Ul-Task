This project is a Node.js practice project to learn modules, file system, OS info, chalk colors, and a mini console calculator.

Project Structure
common → CommonJS version (math.cjs + app.cjs)

es6 → ES6 version (math.js + app.js)

package.json → Project settings and dependencies

history.txt → Saves calculator results

Features

Arithmetic functions: add, subtract, multiply, divide, power, factorial

CommonJS and ES6 module examples

Path info: __dirname, __filename, basename, join

FS module: write/read files, save results

OS module: platform, memory, CPU info

Chalk: colorful console outputs

Mini calculator: prints results in console and saves to history.txt

Run CommonJS version: npm start or start-common

Run ES6 version: npm run start-es6

Check history.txt for saved results
